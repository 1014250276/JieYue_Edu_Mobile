# 默认


### 项目源码git下载
```
git clone https://github.com/1014250276/JieYue_Edu_Mobile.git
```

## 1.安装node_modules模块
```
npm install
```

### 2.启动编译和热重载
```
npm run serve
```

### 3.打包压缩
```
npm run build
```

### 4.运行测试
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
